# System Prompts Leaks

Welcome to the most comprehensive collection of system prompts extracted from popular AI chatbots including Claude, ChatGPT, Gemini, Grok, and many others.

## What is This Collection?

This repository contains extracted system prompts (also known as system messages or developer messages) that define how AI chatbots behave, respond, and interact with users. These prompts are the foundational instructions that shape the personality, capabilities, and limitations of modern AI assistants.

## Why System Prompts Matter

Understanding system prompts helps you:

- **Learn prompt engineering** - See how leading AI companies craft instructions
- **Understand AI behavior** - Discover what shapes AI responses and limitations
- **Research AI safety** - Study how constraints and guidelines are implemented
- **Improve your prompts** - Apply techniques used by professional prompt engineers

<Info>
This collection is maintained for educational and research purposes. All prompts were extracted through documented prompt injection techniques and are publicly shared for transparency.
</Info>

## What's Included

Our collection spans multiple AI platforms and includes:

- Current production system prompts for major AI models
- Tool-specific prompts (Canvas, Memory, Search, Computer Use, etc.)
- API vs. web interface variations
- Personality and persona variations
- Historical/archived versions showing prompt evolution
- Platform-specific implementations (Chrome extensions, desktop apps, Excel add-ins)
